# Quantum Kidz

Quantum Kidz is a parent–teacher communication platform designed for preschools. It provides a single application for parents, teachers, branch heads, and administrators to manage day-to-day school communication and operations.

The project aims to simplify information sharing, attendance management, classroom updates, and administrative workflows while providing a secure and intuitive experience for every user.

## Features

### Parent

* View announcements and classroom updates
* Access photos and activity posts
* Track attendance
* View academic calendar and events
* Receive fee reminders
* Access student information

### Teacher

* Mark student attendance
* Post announcements and classroom activities
* Share photos with parents
* Manage assigned classes

### Branch Head

* Manage teachers and students
* Approve user registrations
* Post branch-wide announcements
* Monitor branch activities

### Super Admin

* Manage branches
* Create and manage classes
* Manage users and permissions
* View platform-wide analytics
